# adonis-ts-cekeran-midun

## Node.js >= 14.15.4

## npm >= 6.0.0

1. clone project

```bash
git clone git@github.com:ayocodingit/crud-adonis-ts.git
```

2. cd crud-adonis-ts

```bash
cd crud-adonis-ts
```

3. run npm install

```bash
npm install
```

4. run node ace migration:run

```bash
node ace migration:run
```

5. run node ace db:seed

```bash
node ace db:seed
```

6. run node ace serve --watch

```bash
node ace serve --watch
```
